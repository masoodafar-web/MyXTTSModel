🎯 چگونه شروع کنیم؟

  1. تازه‌کار هستید؟ ابتدا راهنمای شروع سریع را بخوانید.
  2. می‌خواهید معماری را بفهمید؟ به بخش‌های مدل مراجعه کنید.
  3. با مشکل مواجه شدید؟ راهنمای رفع مشکلات را ببینید.
  4. می‌خواهید مدل را بهینه کنید؟ بخش بهینه‌سازی را مطالعه کنید.

📚 مستندات تکمیلی

علاوه بر این راهنماهای HTML، مستندات زیر نیز در دسترس است:

  • README.md: نمای کلی پروژه و راهنمای سریع
  • docs/FAST_CONVERGENCE_SOLUTION.md: راه‌حل همگرایی سریع
  • docs/GPU_UTILIZATION_FIX_GUIDE.md: راهنمای بهینه‌سازی GPU
  • docs/PLATEAU_BREAKTHROUGH_GUIDE.md: تکنیک‌های شکستن Plateau
  • examples/: نمونه کدهای کاربردی

🚀 شروع سریع

برای شروع فوری آموزش:

# نصب وابستگی‌ها pip install -r requirements.txt # آموزش پایه python3 train_main.py \ --train-data ../dataset/dataset_train \ --val-data ../dataset/dataset_eval # آموزش بهینه شده python3 train_main.py \ --train-data ../dataset/dataset_train \ --val-data ../dataset/dataset_eval \ --optimization-level enhanced \ --enable-gpu-stabilizer

💡 نکات مهم

  • همیشه از آخرین نسخه مستندات استفاده کنید
  • قبل از شروع، سیستم خود را بررسی کنید (GPU, RAM, Storage)
  • با تنظیمات پایه شروع کنید و به تدریج پیشرفته شوید
  • پیشرفت آموزش را با TensorBoard دنبال کنید
  • از checkpoint های منظم برای امنیت استفاده کنید
